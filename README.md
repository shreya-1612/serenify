
# Serenify - Mental Health & Wellness Platform

Serenify is a full-stack mental health and wellness platform designed to help users improve their emotional well-being through guided exercises, mood tracking, therapy booking, progress monitoring, and AI-powered support.

##  Features

###  Authentication

* User Registration
* Secure Login
* JWT Authentication
* Protected Routes

###  Dashboard

* Personalized user dashboard
* Wellness statistics
* Activity tracking
* Progress overview

###  Mood Tracking

* Daily mood logging
* Mood history
* Emotional trend analysis

###  Wellness Exercises

* Guided mental wellness exercises
* Breathing exercises
* Relaxation activities
* Mindfulness practices

###  Daily Checklist

* Daily wellness tasks
* Habit tracking
* Completion monitoring

###  Progress Tracking

* User progress reports
* Achievement monitoring
* Activity insights

###  Chat Support

* Mental wellness chatbot
* Interactive conversations
* Emotional support guidance

###  Therapy Booking

* Browse therapists
* Book appointments
* Session management

###  Subscription & Billing

* Premium plans
* Subscription management
* Payment integration

###  User Profile

* Profile customization
* Personal information management
* Wellness preferences
